# About Text-Conv
This is a my first react project that includes some experiment with the texts you enter.

## Overview
Text-Conv is a React project that allows users to convert their input text into various forms, providing a playful and interactive experience with text manipulation.

## Features
Uppercase Conversion: Transform your text to uppercase.
Lowercase Conversion: Convert your text to lowercase.
Inverse Text: Flip your text in reverse order.
Altering Case: Alternate the case of each character in your text.
